4) All countries Data

This chart gives information about all the countries in the report. It plots the daily case count vs each day in each country. It also annotates the start and end dates of lockdown for each country.

The user can compare between different countries which did/didn't enforce nationwide lockdown.

Conclusion:From these visualications, we can conclude that it is very hard to justify if nationwide lockdown is the most effective way to contain the pandemic. There are other approaches like contact-tracing, mandatory social distancing enforcement etc. There are a few things like resource allocation, pandemic guideline development, border policies wihch can be managed from a central authority but lockdown might not be the most effective step.

TIP 1: You can select the countries you want to highlight in the line chart.

TIP 2: Hover over the chart to see the particular day's date and decreasing order of case count per country on that day.

TIP 3: To clear the chart and see only selected information, click on Reset and Start Time Animation buttons respectively.
