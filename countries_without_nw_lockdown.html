3) Countries Without Nationwide lockdown

This chart gives information about countries which didn't enforce nationwide lockdown. It plots the daily case count vs each day in each country.

For these countries, we see both: an increase as well as decrease in per day case count.

TIP 1: You can select the countries you want to highlight in the line chart.

TIP 2: Hover over the chart to see the particular day's date and decreasing order of case count per country on that day.

TIP 3: To clear the chart and see only selected information, click on Reset and Start Time Animation buttons respectively.
